# `ws-agent pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
ws-agent pairing list whatsapp
ws-agent pairing approve whatsapp <code> --notify
```
